Для одного проекта необходимо было сделать автоматическое создание материала раз в сутки и собственно сам контент нужно было брать со стороннего api. В этом посте я расскажу как создать свои команды для manage.py которые можно использовать в crontab Для примера я буду использовать обновления курса доллара с ЦБ РФ. Сначала опишем нашу модель, она будет максимально простой, будет только значение курса и дата

Здесь как мне кажется все понятно, мы используем библиотеку Grab для того, чтобы взять значение курса и создаем новую запись в базе и по окончании в консоле выводим значение курса. Теперь для использования этого действия мы можем использовать команду updatecurs в manage.py.
